/*
   Файл описания макропримитивов (МП).

   (С) 1997 Шевцов С.П. идея
   (C) 1997 Чумак  А.И. идея, исполнение

   Первая строка в этом файле
   должна быть комментарием не менее 50-ти символов,
   это необходимо для использования в TBL.

-----------------------------------------------------------------------------
Краткое пояснение.

Аргументы:

 Используемые числа:

  номер_состояния или номер_макропримитива:
   целое число в диапазоне от 0 до 65535

  номер_объекта или номер_элемента:
   целое число в диапазоне от 0 до 255

  угол
   число с плавающей точкой в диапазоне от 0.0 до 359.0

 ЦВЕТ:
  BLACK      - чёрный
  BLUE       - синий
  GREEN      - зелёный
  CYAN       - голубой
  RED        - красный
  GOLD       - золотой
  BOLOTO     - грязно-зелёный
  LIGHTGRAY  - светло-серый
  DARKGRAY   - тёмно-серый
  LIGHTBLUE  - светло-синий
  LIGHTGREEN - светло-зелёный
  LIGHTCYAN  - светло-голубой
  LIGHTRED   - светло-красный
  PASTEL     - розовый
  YELLOW     - жёлтый
  WHITE      - белый

 РЕЖИМ:
  SHOW       - показывать изменения в памяти
  NO_SHOW    - не показывать изменения в памяти


Формат команд TBG:

   {команда}{разделитель}[аргумент [...]]

 где:

   {}  - обязательный элемент
   []  - не обязательный элемент
   ... - повторение предыдущего элемента

   команда     - любая команда TBG (см. ниже)
   разделитель - символ двоеточия ":"
   аргумент    - число (числа), ЦВЕТ, РЕЖИМ, "Текст"


Команды TBG, синтаксис:

  вывод всей картинки МП
    outpic_all :
    показать   :           

  вывод всего объекта
    outpic_obj          : номер_объекта
    показать_объект     : номер_объекта

  вывод элемента объекта
    outpic_obj_num      : номер_объекта номер_элемента
    показать_элемент    : номер_объекта номер_элемента

  вывод объекта цветом 
    outpic_obj_color    : номер_объекта ЦВЕТ
    показать_объект_цветом : номер_объекта ЦВЕТ

  вывод элемента объекта цветом 
    outpic_obj_num_color    : номер_объекта номер_элемента ЦВЕТ
    показать_элемент_цветом : номер_объекта номер_элемента ЦВЕТ

  установка цвета объекта 
    setcolor_obj            : номер_объекта ЦВЕТ РЕЖИМ
    установить_цвет_объекта : номер_объекта ЦВЕТ РЕЖИМ

  установка цвета элемента объекта 
    setcolor_obj_num         : номер_объекта номер_элемента ЦВЕТ РЕЖИМ
    установить_цвет_элемента : номер_объекта номер_элемента ЦВЕТ РЕЖИМ

  вывод текста 
    outtext_obj_num          : номер_объекта номер_элемента "ТЕКСТ" ЦВЕТ РЕЖИМ
    установить_текст_объекта : номер_объекта номер_элемента "ТЕКСТ" ЦВЕТ РЕЖИМ

  установка состояния объекта 
    setstate_obj                 : номер_объекта номер_состояния
    установить_состояние_объекта : номер_объекта номер_состояния

  установка состояния элемента объекта 
    setstate_obj_num              : номер_объекта номер_элемента номер_состояния
    установить_состояние_элемента : номер_объекта номер_элемента номер_состояния

  сохраняет текущее состояние МП 
    savestate :
    сохранить_состояние :

  исполнить нужное состояние текущего МП
    execState           : номер_состояния
    исполнить_состояние : номер_состояния         

  вывод элемента объекта с поворотом (только для линий)
    outpic_obj_num_with_rotate    : номер_объекта номер_элемента угол
    показать_элемент_вращением    : номер_объекта номер_элемента угол


Формат записи макропримитива:

// *** Комментарий **********************
 Макро : номер_макропримитива
    Описание : "Описание макропримитива, характеристики и т.д. не более 65535 символов"
    Картинка : "имя файла TBG в формате 8.3"
    Состояние : номер_состояния
       команда TBG : 
       команда TBG : аргумент аргумент аргумент
              . . .
       команда TBG : аргумент аргумент
    Состояние : номер состояния
       команда TBG : аргумент
// **************************************

Примечания:

   1. Состояния под №№ 1, 2 и 3 обязательно должны быть,
      иначе в Редакторе(TBG.EXE) нарисовать этот МП не удастся.

---------------------------------------------------------------------------- */
